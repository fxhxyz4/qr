# 🎨 QR Code Creator

A modern, customizable QR code generator with avatars, gradients, animations, and persistent state — built with pure **HTML, CSS, and JavaScript**.

![GitHub Repo stars](https://img.shields.io/github/stars/fxhxyz4/qr?style=flat-square) ![GitHub forks](https://img.shields.io/github/forks/fxhxyz4/qr?style=flat-square) ![GitHub license](https://img.shields.io/github/license/fxhxyz4/qr?style=flat-square) ![GitHub Pages](https://img.shields.io/badge/deployed-GitHub%20Pages-blue?style=flat-square)

## ✨ Features

- 🔗 Generate QR codes from text or URLs
- 🎭 Avatar support:
  - Emoji mode
  - Image upload mode
- 🌈 20+ gradient background styles
- 💾 Automatic state persistence using `localStorage`
- 📥 Download QR code as PNG
- 🎬 Smooth UI animations powered by Animate.css
- ⚡ No frameworks, no build tools — pure frontend

#### 🛠️ How It Works:

- Enter text or a URL

- Set a username

- Choose an avatar:

- Emoji

- Uploaded image

- Pick a gradient background

- Download the generated QR code as PNG

### All settings are automatically saved in localStorage and restored on page reload.
